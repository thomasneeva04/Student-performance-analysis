# Student Performance Analysis

This project explores student academic performance using data analysis and statistical techniques. The goal is to understand key factors influencing student outcomes and extract meaningful insights from the dataset.

## 📌 Objectives

- Analyze student performance data
- Identify patterns and relationships between variables
- Perform data cleaning and preprocessing
- Generate insights using statistical analysis


## ⚙️ Tools & Libraries

- Python
- NumPy
- Pandas
- Matplotlib


## 📊 Key Concepts Applied

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Data Visualization


## 📁 Project Structure

- `student_performance_analysis.ipynb` → Main analysis notebook  
- `Students Performance Dataset.csv` → Dataset used  


## 📈 Insights

- Identified factors affecting student performance
- Observed trends across different features
- Used statistical summaries and visualizations to interpret data


## 🚀 Learning Outcome

This project helped strengthen my understanding of how statistical analysis and data processing techniques are applied to real-world datasets.
